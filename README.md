## Actividades

### Actividad 1
HTML básico, los primeros encabezados, imágenes y divs.

### Actividad 2
HTML básico, espaciado con `margin` y `padding`.

### Actividad 3
HTML básico, tablas, y propiedades como `background-color` y `font-family`.

### Actividad 4
HTML y CSS básico, **formularios** con la etiqueta `<form>` y método POST.

### Actividad 5
HTML y CSS, usamos propiedad `flex` y etiqueta `<grid>` con un diseño basado en filas y columnas.

### Actividad 6
Empezamos con JavaScript haciendo programas sencillos como calcular el **M.C.D**, mostrar y ocultar una contraseña o mostrar el **día de la semana** dada una fecha.

### Actividad 7 
Un pequeño ejercicio de JavaScript donde al pulsar un botón se genera un número aleatorio entre el 1 y el 6 con forma de dado usando como única imagen un punto negro .svg.
