## Propiedades CSS Básicas

| Propiedad                   | Descripción                                                                                         | Ejemplo                                           |
|-----------------------------|-----------------------------------------------------------------------------------------------------|--------------------------------------------------|
| `color`                     | Establece el color del texto del elemento.                                                          | `color: red;`                                   |
| `background-color`          | Define el color de fondo del elemento.                                                              | `background-color: lightblue;`                  |
| `font-size`                 | Ajusta el tamaño de la fuente del texto.                                                            | `font-size: 16px;`                              |
| `font-weight`               | Establece el grosor de la fuente.                                                                   | `font-weight: bold;`                            |
| `text-align`                | Alinea el texto dentro del contenedor.                                                              | `text-align: center;`                           |
| `margin`                    | Determina el espacio exterior alrededor de un elemento.                                             | `margin: 10px;`                                 |
| `padding`                   | Define el espacio interior entre el contenido y el borde del elemento.                             | `padding: 15px;`                                |
| `border`                    | Establece el estilo, color y grosor del borde de un elemento.                                      | `border: 2px solid black;`                      |
| `display`                   | Controla el tipo de caja de un elemento (block, inline, flex, grid, etc.).                        | `display: flex;`                                |
| `width`                     | Especifica el ancho de un elemento.                                                                 | `width: 100%;`                                  |
| `height`                    | Especifica la altura de un elemento.                                                                | `height: 50px;`                                 |
| `position`                  | Define el método de posicionamiento de un elemento (static, relative, absolute, fixed, sticky).    | `position: absolute;`                            |
| `overflow`                  | Controla cómo se muestra el contenido que desborda el área asignada del elemento.                  | `overflow: hidden;`                             |
| `flex-direction`            | Establece la dirección de los elementos flex (row, column).                                        | `flex-direction: row;`                          |
| `grid-template-columns`     | Define las columnas de una cuadrícula CSS (grid).                                                  | `grid-template-columns: repeat(3, 1fr);`       |
| `box-shadow`                | Añade efectos de sombra a un elemento.                                                              | `box-shadow: 2px 2px 10px grey;`               |
